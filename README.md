# 🍽️ Restaurant Digital Menu

Restaurant Digital Menu adalah aplikasi web berbasis Laravel 12 yang dirancang untuk memudahkan pengelolaan menu restoran secara digital. Aplikasi ini menyediakan panel admin untuk mengelola data menu dan ulasan pelanggan, serta halaman frontend yang memungkinkan pengunjung melihat daftar menu lengkap beserta informasi detail seperti komposisi bahan, nilai gizi, status ketersediaan, dan rating.

## ✨ Fitur Utama

### 👨‍💼 Admin Panel
- Login Admin
- Dashboard dengan statistik menu
- CRUD Menu
- CRUD Review Pelanggan
- Upload Foto Menu
- Status Menu (Ready / Sold Out)
- Search, Filter, Sorting, dan Pagination
- Responsive Admin Dashboard
- SweetAlert2 Notification
- DataTables Integration

### 👥 User
- Login User
- Melihat daftar menu
- Detail menu lengkap
- Melihat rating pelanggan
- Membaca ulasan pelanggan
- Responsive UI

### 🍽️ Menu
- Kategori Food, Beverage, dan Dessert
- Foto Menu
- Harga
- Status Ketersediaan
- Rating
- Komposisi Bahan
- Informasi Kalori
- Protein
- Lemak
- Karbohidrat
- Deskripsi Menu

## 🚀 Instalasi

```bash
git clone https://github.com/USERNAME/restaurant-digital-menu.git

cd restaurant-digital-menu

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate --seed

npm run build

php artisan serve
```
